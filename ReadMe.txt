How to run the program:
1. Download pandas and plot.ly.
2. Place the CSV file on the same folder on the script.
3. Run the script.

Design choices:
1. You can toggle the EEG and EGC lines.
2. You can pan and auto scale the graph.
3. You can export it as a PNG.
4. You can zoom in and zoom out.

Future work:
